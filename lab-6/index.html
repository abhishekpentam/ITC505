Lab 6 Addendum

Problem solved: I built an interactive Bubble Sort tool. The user types any sequence of numbers, and my JavaScript sorts them in ascending order using the Bubble Sort algorithm.

How it works: When you click "Sort Numbers", I read the string from the text input, convert it into an array of integers, and then run a Bubble Sort function in script.js. After every outer pass of the algorithm, I log the intermediate array state in the "Steps" panel so you can watch it get more and more sorted.

Interactivity / events: The Sort button and Random button are both hooked to addEventListener("click", ...). This satisfies the requirement that the page responds to at least one event.

Difficulties: Main issue was cleaning the user input (extra spaces, empty values). I solved it by splitting on commas, trimming each piece, and using Number() so non-numeric values are skipped safely.
